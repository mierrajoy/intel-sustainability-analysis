# intel-sustainability-analysis

#Overview
SQL-based analysis of 150,000+ device records to evaluate energy savings, CO₂ reduction, and sustainability impact of Intel’s device repurposing program.

## Methods
- Analyzed 150,000+ device records using SQL
- Used JOINs, CTEs (WITH), CASE statements, and aggregations
- Examined trends by device type, age, and geographic region
- Calculated total energy savings and emissions reduction

## Key Insights
- Older devices generated higher energy and CO₂ savings per unit
- Regions with higher carbon intensity produced greater environmental impact
- Recommendations were developed to optimize Intel’s repurposing strategy

## Tools
SQL (JOINs, CTEs, CASE, Aggregations)
